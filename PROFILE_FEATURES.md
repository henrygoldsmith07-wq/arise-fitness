# User Profile — Feature Spec (implemented next)

## Header
- Avatar (emoji picker: 👤 ⚔ 👑 🔥 ✨ 🎯 💪 🌌)
- Name
- Editable bio (max 80 chars)
- Rank pill + Rank title
- Level + Streak

## Scores row
- **Fitness Score** — average of STR/END/AGI/VIT (0–100 scaled)
- **Activity Score** — streak + workouts this month
- **Fitness Age** — playful number from onboard strength/mobility/cardio (lower = stronger)
- **Experience Badge** — Novice / Trained / Seasoned / Elite based on total XP

## Dashboards
- **Statistics** — Total XP, Workouts, Quests, Coins, Essence, Boss clears
- **Goals** — Main goal + Weekly focus + Preferred days from onboard

## Collections
- **Achievements** grid (First Quest, 3-Day Streak, 7-Day Streak, Rank Up D, First Boss, 10 Workouts, Skill Advanced, etc.)
- **Personal Records**
- **Favorites** — top exercises & workouts by completion count
- **Calendar** — last 28 days
- **Training History** — last 15 XP events

## Edit
- Tap avatar to change
- Tap bio to edit
- Settings: refresh quests, export, reset
