# ARISE — Expanded User Profile

## Profile features

- **Avatar** — emoji avatar picker (local, no photo upload needed)
- **Bio** — short editable status line
- **Progress level** — current Level + Rank title
- **Experience badge** — derived from total XP / workouts
- **Fitness age** — playful score from strength + mobility + cardio self-checks
- **Activity score** — based on streak + workouts this month
- **Fitness score** — combined attribute average
- **Personal records** — best logged volumes
- **Favorite exercises / workouts** — most completed
- **Achievements** — unlockable badges (first workout, streak milestones, rank ups, boss clears)
- **Workout streak** — consecutive training days
- **Statistics dashboard** — total XP, workouts, quests, coins, essence
- **Goals dashboard** — main goal + weekly focus
- **Calendar** — last 28 days activity
- **Training history** — recent XP events

All data stays in localStorage. Monochrome UI with icons.
