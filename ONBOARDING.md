# ARISE Expanded Onboarding

## Steps implemented

1. **Hunter Name**
2. **Goal selection** — Strength / Endurance / Balanced / Skill
3. **Experience level** — Just starting / Train regularly / Very experienced
4. **Activity level** — Mostly sitting / Lightly active / Active / Very active
5. **Available equipment** — Bodyweight only / Basic (dumbbells/bands) / Full gym
6. **Workout location** — Home / Outdoor / Gym
7. **Preferred duration** — 15 min / 20–30 min / 45+ min
8. **Preferred days** — 2–3 / 4–5 / 6–7 days per week
9. **Body areas to improve** — Upper / Lower / Core / Full body (multi-select)
10. **Strength self-check** — Can you do push-ups / squats / etc.
11. **Mobility self-check** — Comfort reaching / hinging / overhead
12. **Cardio self-check** — How long can you jog or climb stairs
13. **Motivation** — Why you want to train

## How answers are used
- Experience + strength/mobility/cardio answers set starting attribute values
- Equipment + location influence which workouts are recommended
- Duration + days shape weekly challenge targets
- Goal + focus areas bias daily quests and skill path suggestions

## Intentionally light
- No medical diagnosis questions
- No heavy weight-loss framing (age-appropriate)
- All data stays local in the browser
