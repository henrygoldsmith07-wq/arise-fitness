# ARISE — Level Up In Real Life

A clean, gamified fitness web app inspired by Solo Leveling and top fitness apps.

**Live demo:** Open `index.html` or enable GitHub Pages on this repository.

## Features

- **Daily Quests** & Weekly Challenge
- **Skill Paths** with progressive overload (practice → advance)
- Form tips + common mistakes
- **Boss Raids**
- Rest timer, Personal Records, Activity calendar
- Structured Programs (Foundation, Strength Builder, Hunter Path)
- Rank system: Novice Hunter → Shadow Commander
- Clean, minimal UI
- Fully offline — all data stays in your browser

## How to use

1. Open `index.html` in any modern browser
2. Complete the short onboarding
3. Start with Daily Quests
4. Practice skills and level up

No account required. Everything is stored locally.
